# Job Application Tracker

A Python-based application for managing and tracking job applications using SQL database operations.

## Features

* Add new job applications
* View all applications
* Search applications by company name
* Update application status
* Delete applications
* Persistent data storage using SQLite

## Technologies Used

* Python 3
* SQLite
* SQL
* VS Code

## Database Operations

The application implements CRUD operations:

* Create (INSERT)
* Read (SELECT)
* Update (UPDATE)
* Delete (DELETE)

## Project Structure

* `job_tracker.py` – Main application
* `database_setup.py` – Database and table creation
