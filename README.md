# Job Application Tracker

A simple Python application for managing and tracking job applications.

## Features

* Add new job applications
* View all applications
* Search applications by company name
* Track application status

## Technologies Used

* Python 3
* VS Code

## Purpose

This project was built to help job seekers organize and track their job applications efficiently.

## Future Improvements

* SQL Database Integration
* Update Application Status
* Delete Applications
* Export Data to Excel
